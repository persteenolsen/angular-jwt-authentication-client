Description

This Angular SPA Client work together with a secure ASP.NET Core 2.2 Web API using JWT Authentication

Last Updated
  • 13-07-2020
Main functionality
  • A User can perform a login using JWT Authentication with the Baerer scheme
  • A User have access t secure pages after successfully login
Module Bundler
  • Manuel configuration of the Webpack 4 serving as a module bundler
Security
  • JWT Authentication with the Baerer scheme
  • The user will have access to the secure endpoints of the Web API passing login and using the JWT
  • The endpoints of the Web API are secured by JWT
  • The Web API is secured by CORS allowing only the subdomain containing the SPA client making HTTPS request
The Angular Single Page Application - The frontend
  • Angular 7
  • TypeScript
  • Less
  • HTML and CSS
  • Mainly JavaScript ES5, ES6 and ES7
  • Traditionel Bootstrap 4 by CDN for the responsive design
ASP.NET Core 2.2 Web API - The backend
  • ASP.NET Core 2.2 Web API
Hosting
  • The Angular SPA client is hosted at a traditionel Webhotel on an IIS
  • The Web API is hosted at a traditionel Webhotel on an IIS
Text Editor
  • Visual Studio Code as Text Editor and Visual Studio as IDE
Testing
  • Test Driven Developement and Unit Testing all the way developing the system
